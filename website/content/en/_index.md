{{% blocks/lead color="primary" %}}
BobaMixer is a powerful CLI tool that manages multiple AI providers, tracks costs, and optimizes your AI workload routing.

**Key Features**: Multi-provider support, intelligent routing, real-time budget tracking, and comprehensive usage analytics.
{{% /blocks/lead %}}


{{% blocks/section color="dark" type="row" %}}
{{% blocks/feature icon="fa-lightbulb" title="Intelligent Routing" %}}
Route prompts to the best AI provider based on context, cost, and performance with epsilon-greedy exploration.
{{% /blocks/feature %}}


{{% blocks/feature icon="fa-chart-line" title="Budget Tracking" %}}
Track costs at global, project, and profile levels with daily/monthly limits and real-time alerts.
{{% /blocks/feature %}}


{{% blocks/feature icon="fab fa-github" title="Multi-Provider Support" %}}
Unified interface for HTTP APIs, command-line tools, and MCP (Model Context Protocol) servers.
{{% /blocks/feature %}}
